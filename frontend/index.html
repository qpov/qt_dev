Ваш надежный помощник для управления голосовыми каналами в Discord

Что умеет бот?

    Создаёт временные голосовые каналы с именем пользователя при подключении

    Автоматически удаляет созданные каналы, если они становятся пустыми

    Простое управление настройками через удобный веб-интерфейс

    Работает с несколькими серверами одновременно

    Не трогает существующие каналы на сервере

Как начать использовать?

    Войдите через Discord, нажав на кнопку ниже

    Настройте бота на вашем сервере

    Укажите канал, к которому должны подключаться пользователи для создания временных каналов

    Наслаждайтесь автоматизацией и удобством!

Начать прямо сейчас!
